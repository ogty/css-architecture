## Name Methods

- sub-title: Kebab case or Hyphen case
- sub_title: Snake case
- subTitle: Camel case or Lower camel case
- SubTitle: Upper case or Pascal case

## 4 Goals

1. Predictable
2. Reusable
3. Maintainable
4. Expandable
